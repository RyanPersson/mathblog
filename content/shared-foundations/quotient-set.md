---
title: "Quotient set"
description: "The set A/∼ of equivalence classes of A under an equivalence relation ∼"
---

Let $\sim$ be an {{< knowl id="equivalence-relation" text="equivalence relation" >}} on a {{< knowl id="set" text="set" >}} $A$. The **quotient set** $A/{\sim}$ is the set of all {{< knowl id="equivalence-class" text="equivalence classes" >}}:
$$
A/{\sim}=\{[a]: a\in A\}.
$$

There is a canonical {{< knowl id="surjective-function" text="surjective" >}} {{< knowl id="function" text="function" >}} $\pi\colon A\to A/{\sim}$, called the quotient map, defined by $\pi(a)=[a]$.

**Examples:**
- For congruence modulo $n$ on $\mathbb{Z}$, the quotient set $\mathbb{Z}/{\sim}$ is the set of residue classes modulo $n$ (often written $\mathbb{Z}/n\mathbb{Z}$ as a set).
- If $\sim$ is equality on $A$, then $A/{\sim}$ can be identified with $A$ via $a\mapsto [a]=\{a\}$.
