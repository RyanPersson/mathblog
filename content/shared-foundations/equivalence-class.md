Let $\sim$ be an {{< knowl id="equivalence-relation" text="equivalence relation" >}} on a {{< knowl id="set" text="set" >}} $A$, and fix $a\in A$. The **equivalence class** of $a$ is the subset
$$
[a]=\{x\in A : x\sim a\}\subseteq A,
$$
so $[a]$ is a {{< knowl id="subset" text="subset" >}} of $A$.

Distinct equivalence classes are disjoint, and the set of all classes forms a {{< knowl id="partition" text="partition" >}} of $A$.

**Examples:**
- If $\sim$ is equality, then $[a]=\{a\}$.
- For congruence modulo $3$ on $\mathbb{Z}$, the class of $1$ consists of all integers of the form $3k+1$.
- For $x\sim y$ iff $x-y\in\mathbb{Z}$ on $\mathbb{R}$, each class is a translate of $\mathbb{Z}$ inside $\mathbb{R}$.
