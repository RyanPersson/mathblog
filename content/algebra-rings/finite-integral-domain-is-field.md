---
title: "Finite integral domains are fields"
description: "A finite integral domain has multiplicative inverses for all nonzero elements."
---

**Finite integral domains are fields**: If $D$ is a finite integral domain, then $D$ is a field.

Using {{</* knowl id="cancellation-integral-domain" text="cancellation" */>}} in an {{</* knowl id="integral-domain" text="integral domain" */>}}, the map $x\mapsto ax$ is injective for $a\neq 0$; finiteness forces it to be surjective, so $a$ is a {{</* knowl id="unit" text="unit" */>}}. Hence every nonzero element is invertible and the ring is a {{</* knowl id="field" text="field" */>}}.
