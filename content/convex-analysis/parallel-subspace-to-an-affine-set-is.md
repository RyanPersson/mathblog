---
title: "Parallel Subspace to an Affine Set is Ω−Ω"
description: "Every nonempty affine set is parallel to a unique subspace L=Ω−Ω."
---

Let $X$ be a {{< knowl id="vector-space" section="shared-linear-algebra" text="vector space" >}} and let $\Omega\subset X$ be a nonempty {{< knowl id="affine-set" text="affine set" >}}.

**Proposition**: The set $\Omega$ is parallel to a unique {{< knowl id="linear-subspace" text="linear subspace" >}} $L\subset X$, and this subspace is
$$
L=\Omega-\Omega:=\{u-v\mid u,v\in\Omega\}.
$$

**Context:**
The subspace $\Omega-\Omega$ captures the "direction" of the affine set. It is the appropriate linear object used to define {{< knowl id="hyperplane" text="hyperplanes" >}} as affine sets of {{< knowl id="codimension" text="codimension" >}} one.
