---
title: "Continuous function on a compact set is uniformly continuous"
description: "On compact domains, continuity automatically upgrades to uniform continuity"
---

**Corollary (Heine–Cantor)**: Let $(X,d_X)$ be a {{< knowl id="compact-set" text="compact" >}} {{< knowl id="metric-space" text="metric space" >}} and let $(Y,d_Y)$ be a metric space. If $f:X\to Y$ is {{< knowl id="continuity-on-a-set" text="continuous" >}}, then $f$ is {{< knowl id="uniform-continuity" text="uniformly continuous" >}} on $X$.

**Connection to parent theorem**:
This is precisely the {{< knowl id="heine-cantor-theorem" text="Heine–Cantor theorem" >}}.
